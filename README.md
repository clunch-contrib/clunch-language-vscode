# Clunch Language VSCode

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/clunch-contrib/clunch-language-vscode/issues)，欢迎参与维护！

## 打包使用

我们当前只提供本地安装，没有发布到应用市场。

```bash
npm i vsce -g
```

安装好打包工具后，直接运行：

```bash
vsce package
```

生成好的vsix文件从扩展的右上角选择``` Install from VSIX ```安装即可！
